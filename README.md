# DataCase-Flavours

**Project Overview**
This project demonstrates an end-to-end ETL pipeline that transforms raw CSV data into a dimensional data warehouse, ready for analytics and dashboard reporting.
Highlights:
Multi-source raw data ingestion


Automated data quality checks (nulls, duplicates, timestamps)


Dimensional model (Dimensions & Facts)

**Summary**

This repository demonstrates a complete ETL pipeline for Taste Data Analytics:

Raw → Processed → Warehouse (Dimensions & Facts) → Analytics-ready CSVs

It ensures data quality, traceability, and usability for analytics dashboards.
